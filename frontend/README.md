# DZ-CarPool Frontend

Frontend React pour la plateforme de covoiturage DZ-CarPool.

## Démarrage

### Avec Docker

```bash
docker-compose up frontend
```

### Sans Docker

```bash
npm install
npm start
```

## Scripts disponibles

- `npm start` - Lance l'app en mode développement
- `npm test` - Lance les tests
- `npm run build` - Build de production
- `npm run eject` - Eject de create-react-app (irréversible)

## Variables d'environnement

Voir `.env` pour la configuration.
